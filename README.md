  <br />

  <table>
    <tr>
      <th>Learning Goals</th>
      <td>
        <ul>
          <li>
            Get familiar with advance filtering techniques, handle multiple filters effectively and
            update the URL (search params).
          </li>
          <li>
            React: Enhance my knowledge (complex state management, custom hooks, separation of
            concerns, MVVM).
          </li>
          <li>TypeScript: Handle more complex types.</li>
          <li>CSS conventions: Learn an follow BEM methodology for clean and reusable styles.</li>
        </ul>
      </td>
    </tr>
    <tr>
      <th>Features</th>
      <td>
        <ul>
          <li>
            <b>Dynamically Multiple Filtering</b>: Use multiple filters to filter a dataset by
            different categories.
          </li>
          <li>
            <b>Filter Status Managment</b>: See which filters are "available" and which are
            currently "active".
          </li>
          <li>
            <b>Interactive Filter Selection</b>: Click to move filters between "available" and
            "active" lists.
          </li>
          <li>
            <b>Adaptive Remaining Filters</b>: Available filters show only those options that the
            user can still choose from.
          </li>
          <li>
            <b>URL Sync</b>: Active filters are reflected in the browser URL. Simply share the
            current filter combination, or paste a URL to revisit a specific set of filters.
          </li>
          <li><b>Reset Filters</b>: Clear all active filters and reset the URL to start over.</li>
        </ul>
      </td>
    </tr>
    <tr>
      <th>Technologies</th>
      <td>
        <ul>
          <li>TypeScript</li>
